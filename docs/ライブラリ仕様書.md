### はじめに
本書はPythonの麻雀計算ライブラリ「mahjong」の仕様を記したドキュメントである  
尚、ソースを基に書き起こした非公式であり誤りの可能性がある

### 計算について
- 計算結果を得るにはHandCalculatorを使用する
- mahjong.hand_calculating.handよりHandCalculatorをインポートすることで使用可能となる
- estimate_hand_valueメソッドを使用する
  - 引数は以下の通り
    - tiles: 和了形
    - win_tile: 和了牌
    - melds: 鳴き
    - dora_indicators: ドラ表示牌
    - config: オプション

### 和了形・和了牌について
- 和了形・和了牌はTilesConverterを使用する
  - 和了形には和了牌を含む14枚を渡す必要がある
- 種類
  - 萬子の場合: man
    - 1～9を指定
  - 筒子の場合: pin
    - 1～9を指定
  - 索子の場合: sou
    - 1～9を指定
  - 字牌の場合: honors
    - 1～7を指定
      - 1:東 2:南 3:西 4:北 5:白 6:發 7:中
- mahjong.tileよりTilesConverterをインポートすることで使用可能となる


```
[sample]
    from mahjong.hand_calculating.hand import HandCalculator
    from mahjong.tile import TilesConverter

    calculator = HandCalculator()
    # 和了形
    tiles = TilesConverter.string_to_136_array(man='123', pin='22567', sou='123456')
    # 和了牌
    win_tiles = TilesConverter.string_to_136_array(sou='5')[0]
    # 鳴き
    melds = None
    # ドラ表示牌
    dora_indicators = None
    # オプション
    config = None
    # 計算
    result = calculator.estimate_hand_value(tiles, win_tiles, melds, dora_indicators, config)
```

### 鳴きについて
- 計算結果に鳴きを含めるにはmeldを使用する
- 鳴きの種類
    - チーの場合: CHI
    - ポンの場合: PON
    - カンの場合: KAN
    - 加槓の場合: CHANKAN
    - 抜きの場合: NUKI
- mahjong.meldよりMeldをインポートすることで使用可能となる
- Meldメソッドを使用する
  - 引数は以下の通り
    - meld_type: 鳴きの種類
    - tiles: 自身が鳴いた牌
    - opened: 明槓か暗槓か
    - called_tiles: 他家が鳴いた牌
    - who: 誰が鳴いたか
    - from_who: 誰から鳴いたか

```
--- チー ---
[sample](役:三色同順)
    from mahjong.hand_calculating.hand import HandCalculator
    from mahjong.tile import TilesConverter
    from mahjong.meld import Meld
    
    calculator = HandCalculator()
    # 和了形
    tiles = TilesConverter.string_to_136_array(man='12377', pin='123456', sou='123')
    # 和了牌
    win_tiles = TilesConverter.string_to_136_array(sou='3')[0]
    # 鳴き(単数指定)
    melds = [Meld(Meld.CHI, TilesConverter.string_to_136_array(man='123'))]
    # 鳴き(複数指定)
    # melds = [
    #    Meld(Meld.CHI, TilesConverter.string_to_136_array(man='123')),
    #    Meld(Meld.CHI, TilesConverter.string_to_136_array(pin='456')),
    # ]
    # 計算
    result = calculator.estimate_hand_value(tiles, win_tiles, melds)
    # ドラ表示牌
    dora_indicators = None
    # オプション
```

### オプションについて
- 計算結果にオプションを含めるにはconfigを使用する
- 以下が設定可能なオプションである
  - HandConfig(mahjong.hand_calculating.hand_configよりHandConfigをインポートすることで使用可能となる。自風および場風についてはmahjong.constantsよりEAST, SOUTH, WEST, NORTHをインポートすることで使用可能となる)
    - 自摸: is_tsumo
      - 初期値: 無効
    - 立直: is_riichi
      - 初期値: 無効
    - 一発: is_ippatsu
      - 初期値: 無効
    - 嶺上開花: is_rinshan
      - 初期値: 無効
    - 槍槓: is_chankan
      - 初期値: 無効
    - 海底摸月: is_haitei
      - 初期値: 無効
    - 河底撈魚: is_houtei
      - 初期値: 無効
    - ダブル立直: is_daburu_riichi
      - 初期値: 無効
    - 流し満貫: is_nagashi_mangan
      - 初期値: 無効
    - 天和: is_tenhou
      - 初期値: 無効
    - 人和: is_renhou
      - 初期値: 無効
    - 地和: is_chiihou
      - 初期値: 無効
    - 自風: player_wind
      - 初期値: 無効
    - 場風: round_wind
      - 初期値: 無効
  - OptionalRules(mahjong.hand_calculating.hand_configよりHandConfig, OptionalRulesをインポートすることで使用可能となる)
    - 喰い断を有効化: has_open_tanyao
      - 初期値: 無効
    - 赤ドラを有効化: has_aka_dora
      - 初期値: 無効
    - ダブル役満を有効化: has_double_yakuman
      - 初期値: 有効
    - 数え役満の設定: kazoe_limit
      - 初期値: 無効
    - 切り上げ満貫を有効化: kiriage
      - 初期値: 無効
    - 鳴いて和了ると2符加算: fu_for_open_pinfu
    ※麻雀のルール上鳴き自摸のみ2符加算
      - 初期値: 有効
    - 平和自摸を無効化: fu_for_pinfu_tsumo
      - 初期値: 無効
    - 人和を役満化: renhou_as_yakuman
    ※無効の場合: 5翻
      - 初期値: 無効
    - 大車輪自摸を有効化: has_daisharin
    ※無効の場合: 12翻{門前自摸、平和、断么九、二盃口、清一色}(ドラなし)
      - 初期値: 無効
    - 大車輪ロンを有効化: has_daisharin_other_suits
    ※無効の場合: 11翻{平和、断么九、二盃口、清一色}(ドラなし)
      - 初期値: 無効
